Telegram-бот для отработки навыков решения квадратных уравнений

Описание проекта: Telegram-бот является справочником теории и тренажером для отработки пользователем своих знаний. Бот содержит три уровня сложности решения уравнений и поддерживает статистику у пользователя.

Цель проекта: : разработать Telegram-бота, который помогает пользователям освежить теоретические знания по квадратным уравнениям, отработать их решение на практике с помощью заданий разной сложности и отследить результаты с помощью базовой статистики.

Запуск локально

1. Установите Python 3.10
2. Установите зависимости:
pip install -r requirements.txt
3. Установи pre-commit хуки:
pre-commit install
4. Запустите бота
python main.py

Запуск в Docker

1. В командной строке перейдите в папку проекта
2. Выполните:
docker build -t my-bot .
docker run --rm my-bot

Конфигурация (переменные окружения)

Бот использует BOT_TOKEN для конфигурации: Токен Telegram-бота встроен в код бот.ру

Примеры использования

Бот имеет несколько команд:
/start - Запуск бота
/theory - Материал и теория
/practice - Тренажер
Содержит кнопки меню: Теория, Тренажер, Статистика.
Выходные данные:
1. Базовый уровень (пошаговое решение уравнений)
    Нажатие на инлайн-кнопки "Шаг 1", "Шаг 2" и т.д.
    Подробное решение с формулами и вычислениями
2. Средний уровень (выбор ответа из предложенных вариантов)
    Выбор 1 из 3 вариантов ответов-кнопок
    Если верный ответ был выбран, будет показан итог уравнения, а если нет, то выйдет сообщение и картинка с полным решением квадратного уравнения
3. Профи уровень (самостоятельный ввод ответа)
  Текстовый ответ (например: "2, 3") в чат
  Проверка корректности ответа и вывод сообщения об этом
